# Microsoft Exchange

## Virtual Directory

在 Exchange 中，Virtual Directory 是指在 IIS 中为 Exchange 服务创建的虚拟目录。它们用于处理不同类型的客户端请求，例如 OWA (Outlook Web Access)、ECP (Exchange Control Panel) 等。每个虚拟目录都有特定的 URL 路径和配置，用于提供相应的服务。

Exchange 总共有八大虚拟目录：

Autodiscover, ECP, EWS, MAPI, OAB, OWA, EAS, Powershell

- Autodiscover：提供自动发现服务，帮助客户端自动配置连接设置。
- ECP (Exchange Control Panel)：提供基于 Web 的管理界面。
- EWS (Exchange Web Services)：提供基于 Web 的 API，用于访问 Exchange 数据。
- MAPI (Messaging Application Programming Interface)：提供基于 Web 的 MAPI 访问。
- OAB (Offline Address Book)：提供离线地址簿服务。
- OWA (Outlook Web Access)：提供基于 Web 的邮件访问。
- EAS (Exchange ActiveSync)：提供基于 Web 的 ActiveSync 访问。
- PowerShell：提供基于 Web 的 PowerShell 访问。
