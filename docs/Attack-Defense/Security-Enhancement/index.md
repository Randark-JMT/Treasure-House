# 安全加固

根据著名的木桶效应理论，一整套体系的安全强度，不在于防御最强的那几个部分，而在于其最不安全的那一个组件（最短的那一块木板），不断提升最短的那一块木板的长度，便可以提升一整个体系的安全程度。
