## Docker CFS

CFS，即三层内网靶场，传统部署方式为使用多台虚拟机以及三个虚拟网络进行部署，消耗的计算资源较大。现在得益于 Docker、Kubernetes 等容器化技术，使得单台靶机在可行的情况下转为用容器进行部署，大大降低了对资源的消耗，使得 CFS 靶场可以实现在性能较为薄弱的设备上进行部署。
